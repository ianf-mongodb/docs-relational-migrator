===========================
What is Relational Migrator
===========================

.. introduction::

   MongoDB Relational Migrator is a tool to help you migrate relational workloads to MongoDB. Relational Migrator allows you to:

   - Transform your relational data and schemas to MongoDB.
   - Plan and schedule your MongoDB migrations.
   - Migrate at scale and resiliency with Kafka support.

   .. button:: Install Relational Migrator
      :uri: http://docs-migrator.s3-website-us-east-1.amazonaws.com/main/Installation/

.. kicker:: What You Can Do

Control Your Migration To MongoDB
---------------------------------

.. tabs::

   .. tab:: Migrate To MongoDB
      :tabid: Migrate To MongoDB

      Relational Migrator allows you to to transfer your relational data and schemas to MongoDB.
      Using snapshot or continuous sync jobs you can choose a migration strategy that is right for you.
      Relational Migrator supports migrations from Long-term support versions of the following databases:

      - MySQL
      - Oracle
      - PostgreSQL
      - SQL Server

   .. tab:: Model Your Data
      :tabid: Model Your Data

      .. procedure::

         .. step:: Import your data

            Relational Migrator allows you to define customized mapping 
            rules that convert your table row data to MongoDB documents. 
            Relational Migrator helps you transform your data models so 
            that they work best for your applications.

            `To learn more, see mongoimport <https://www.mongodb.com/docs/database-tools/mongoimport/>`__
      
      .. image:: /images/Mapping-Rules.png
         :alt: Mapping Rules

   .. tab:: Visualize Your Migration
      :tabid: Visualize Your Migration

      Relational Migrator allows you to view both your source 
      relational and destination MongoDB data models. 
      Using Relational Migrator diagraming helps you edit your 
      data model and visualize your migration in real time.

      .. image:: /images/Schema-Models.png
         :alt: Schema Models

   .. tab:: Deploy and scale your database
      :tabid: deploy

      .. procedure::

         .. step:: Model your data

            Relational Migrator Code Generation allows you to convert 
            your existing data layer application code. With Code Generation,
            you can automatically transform application code that was 
            written for connecting with relational data models to 
            connect to MongoDB data models.

      .. code-block:: javascript

         /**
         * Generated by MongoDB Relational Migrator 
         * https://www.mongodb.com/products/relational-migrator 
         * Collection: actor
         * Language: JavaScript
         * Template: Mongoose
         * Generated on 5/1/23
         */
         import mongoose from "mongoose";
         const Schema = mongoose.Schema;
         const model = mongoose.model;
         export const actor_Mongoose = new Schema({
            "_id": mongoose.ObjectId,
            "first_name": String,
            "last_name": String,
            "last_update": String,
         }, { collection: "actor" })
         export const actor_MongooseModel = model("actor_Mongoose", actor_Mongoose);
