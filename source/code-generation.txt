===============
Code Generation
===============

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Relational Migrator allows you to convert your code to MongoDB syntax. 
You can use the following features within the 
:guilabel:`Code Generation` tab to convert your code:

.. list-table::
   :header-rows: 1
   
   * - Feature 
     - Supported Conversion 

   * - Query Converter 
     - 
      - SQL queries 
      - SQL views
      - Stored procedures 
   * - Generate App Code
     - 
      - Language specific application code
