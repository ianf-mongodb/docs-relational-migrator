===============
Code Generation
===============

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Relational Migrator can assist you in converting both your SQL and 
application code to MongoDB. By utilizing Relational Migrator's code 
generation features, you can streamline the migration process and 
save time and effort.

Use Cases
---------

You can use the following features within the 
:guilabel:`Code Generation` tab to convert your legacy code:

.. list-table::
   :header-rows: 1
   
   * - Feature 
     - Supported Conversions 

   * - Query Converter 
     - 
      - SQL queries 
      - SQL views
      - Stored procedures 
   * - Generate App Code
     - 
      - ``C#``, ``Java``, ``JavaScript``, and ``JSON`` application code.

.. note::

   Query Converter is in private preview, for details on how to sign up
   see :ref:`rm-query-converter`.
