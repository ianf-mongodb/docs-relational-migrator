==========================
Query Converter Test Bench
==========================

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Use Query Converter Test Bench to test converted SQL statements, 
views, and stored procedures. Relational Migrator uses the connections 
defined for your source and destination databases and can run the 
original and converted queries so that you can compare the results.
