================
Project Overview
================

When you open a project in Relational Migrator, the project dashboard displays.

.. image:: /img/projects/project-overview-annotated.png
   :alt: Project Overview Screen

You can manage many aspects of your Relational Migrator project from this dashboard.

.. list-table::
   :header-rows: 1

   * - UI Element
     - Description
   * - Top Navigation Bar
     - Project navigation, settings, primary application navigation, get help menu
   * - Left pane
     - Relational and MongoDB schema models, filter options, view modes
   * - Diagram toolbar
     - selection, layout, foreign keys, diagram tabs
   * - Diagrams
     - Relational and MongoDB diagrams, zoom control, mini map
   * - Right pane 
     - Viewing and modifying mapping rules
